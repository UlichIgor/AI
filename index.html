Ласкаво просимо до GitHub Copilot.

GitHub Copilot — це інтелектуальний помічник для розробників, який пропонує кодові фрагменти та допомагає вам швидше писати код.

GitHub Copilot — це інструмент, який використовує штучний інтелект для автоматичного створення коду на основі ваших коментарів та контексту вашого проекту.

Ось як можна розпочати. Тепер у вас є доступ до GitHub Copilot у VS Code і github.com безкоштовно як частина вашого облікового запису користувача GitHub — і це включає підтримку моделей від OpenAI та Anthropic..
